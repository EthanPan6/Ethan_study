## 按位（bitwise）NOT 技巧

这里使用的一个老技巧是 bitwise NOT ~ 运算符。它将数字转换为 32-bit 整数（如果存在小数部分，则删除小数部分），然后对其二进制表示形式中的所有位均取反。

实际上，这意味着一件很简单的事儿：对于 32-bit 整数，~n 等于 -(n+1)。
```js
alert( ~2 ); // -3，和 -(2+1) 相同
alert( ~1 ); // -2，和 -(1+1) 相同
alert( ~0 ); // -1，和 -(0+1) 相同
alert( ~-1 ); // 0，和 -(-1+1) 相同
```
正如我们看到这样，只有当 n == -1 时，~n 才为零（适用于任何 32-bit 带符号的整数 n）

适用于当`返回值===-1`的方法的简写
```js
let str = "Widget";

if (str.indexOf("Widget") != -1) {
    alert("We found it"); 
}

if (~str.indexOf("Widget")) {
  alert( 'Found it!' ); // 正常运行
}
```

#### `0`的判断
正负数都是`true`，`0`是false，`‘0’`是字符串，所以为`true`
```js
Boolean(0)//false
Boolean(-0)//false
Boolean(+0)//false
Boolean(-1)//true
Boolean(+1)//true
Boolean('0')//true
```
